# T2-IA
Usamos um dataset de falacias logicas feito por nós, estudamos o que cada uma falacia logica é e pegamos alguns 
exemplos genéricos e reais, alguns como debates, videos no youtube e até tweets. Nesses exemplos reais nós mesmos
classificamos as falacias encontradas neles. Usamos de exemplo adicional pedido pelo professor o zero-shot-chain-of-thought.
Todos testes feitos no modelo Gemini 2.5 flash por chamada de API gratuita disponibilizada.
